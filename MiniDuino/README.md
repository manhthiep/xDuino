MiniDuino
======

MiniDuino - Arduino-compatible board with minimal design.
This board includes FTDI Basic for programming over USB.

It's a stripped down version of SparkFun's ProtoSnap.

Original design: 
[SparkFun's Arduino Pro Mini 328](https://www.sparkfun.com/products/11113)
[SparkFun's ProtoSnap - Pro Mini] (https://www.sparkfun.com/products/10889)
